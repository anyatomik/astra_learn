# Для Debian дистрибутивов

Примеры дистрибутивов: Astra Linux, Ubuntu, Kali Linux

Шаг 1: Скачайте .deb пакет по ссылке нажмите для перехода в окно загрузок

Шаг 2: Проверьте путь файла, когда он скачается, запомните путь и имя файла

Шаг 3: Откройте терминал

Шаг 4: Введите команду 

cd ~/Download/

Вместо “Downloads”, введите пть, который вы зпомнили из пункта 2

Шаг 5: Введите команду 

sudo dpkg -i filename.deb

Вместо “filename”, введите имя файла, который вы запомнили из пункта 2
